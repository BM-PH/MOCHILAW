01 — STAFF 교육 매뉴얼

서버내 실무 교육

K - 관리 - 관리자 메뉴 각 기능

1-1. #고유번호 살리기 (유저 요청이 있을 시)

1-2. #여러명 끌고가기 (웬만하면 사용X)

1-3. [관리자] 무적모드 (rp 및 좀비존, 사냥터 이용 시 사용 불가)

1-4. 보안 오브젝트 삭제 (핵접속시 사용 이외 사용X)

K - 관리 - 이동 - 좌표 관련 기능

2-1. 좌표 확인 (팩션요청으로 차량 비콘 및 옷장 비콘 생성요청시)

2-2. 좌표로 텔레포트 (이용X)

K - 관리 - 이동 기능

3-1. [스태프] 노클립 (이용X — F2 노클립 사용)

3-2. 관리자 사유지 (수영하고 싶을 때 이용)

3-3. 혼자있고 싶은 곳 (우울하고 힘들 때 이용)

3-4. 관리자 술집 (2층에서 스태프 스킨 이용 가능)

K - 관리 - 역할 관리 기능

4-1. 권한 추가 (팩션 직급 추가)

4-2. 권한 삭제 (팩션 직급 제거)

K - 관리 - 밴 & 추방 기능

5-1. [관리자] 구금 — 관리자 구금 넣는 것

5-2. [관리자] 구금해제 — 관리자 구금 해제

5-5. 플레이어 게임추방 (웬만하면 F7 눌러서 이용)

5-6. 차단 (팩션해체 및 차단해야 되는 유저들에게 이용)

5-7. 차단(해제) — 차단 해제 시 이용

5-8. [추노기록] 작성 (관구 및 물품회수 등 유저가 없을 때)

5-9. [추노기록] 삭제 — 완료 시 해제 [꼭 삭제하기]

디스코드 실무 교육

모든 디스코드 채널은 수시로 확인바랍니다.

1. 뉴비재인증: 서버목록 / 인게임화면 / 별명양식 충족 시 권한 지급

2. 권한삭제: 무조건 팩션쿨타임 권한 지급

관리팀 디스코드

1. 물품회수 완료 시 [물품회수] 채널 작성

4. 개인사정 미접속 시 [휴가신청] 필수 작성

5. 출퇴근 시 /출퇴근 명령어 사용

6. 사건처리 완료 시 /사처등록 명령어 사용

팩션 디스코드

팩션 디스코드방은 담당 관리진이 아닐 경우 할 일이 없으나 가끔 패치요청이 올라올 경우 해당 요청만 관리팀 디스코드 패치요청에 올리기

고객센터 디스코드

1. 문의가 시작되었을 경우

안녕하세요, 모찌서버 운영팀 OOO입니다 무엇을 도와드릴까요?

2. 문의 대응 도중 모르는 상황이 있을 경우

잠시만 기다려주시면 답변 도와드리겠습니다.

위에 채팅 작성 후 (! @모찌 운영팀 태그)

3. 오래된 문의를 늦게 답변할 경우

답변이 늦어져 죄송합니다. <해당 상황에 맞는 답변>

4. 문의 종료할 때

추가 문의사항 있으신가요?

작성 이후 문의사항이 없을 때

문의 종료 하겠습니다.

이후 (.문의종료) 작성
